Get Started

Get started with the Sanas Self Hosted SDK by downloading and running your first model.

1. Download the SDK
# Install the Sanas SDK
pip install sanas-sdk

# Or download directly
curl -O https://sdk.sanas.ai/latest/sanas-sdk.tar.gz
tar -xzf sanas-sdk.tar.gz
2. Initialize and run a model
from sanas_sdk import SanasClient

# Initialize the client with your API key
client = SanasClient(api_key="YOUR_API_KEY")

# Run accent translation
result = client.translate_accent(
    audio_file="input.wav",
    target_accent="us-english"
)

# Save the output
result.save("output.wav")
print("Translation complete!")
